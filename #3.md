# –°–∏–ª–∞ —É–ø—Ä—É–≥–æ—Å—Ç–∏. –ó–∞–∫–æ–Ω –ì—É–∫–∞

$F_{—É–ø—Ä}=-F=-E\cdot\frac{\Delta x}{x}\cdot S$

- –≥–¥–µ –ï ‚Äì –∫–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç –ø—Ä–æ–ø–æ—Ä—Ü–∏–æ–Ω–∞–ª—å–Ω–æ—Å—Ç–∏, –Ω–∞–∑—ã–≤–∞–µ–º—ã–π –º–æ–¥—É–ª–µ–º —É–ø—Ä—É–≥–æ—Å—Ç–∏ –∏–ª–∏ –º–æ–¥—É–ª–µ–º –Æ–Ω–≥–∞. –ï–¥–∏–Ω–∏—Ü–∞ –∏–∑–º–µ—Ä–µ–Ω–∏—è –ù/–º2.

$\sigma=\frac{F{—É–ø—Ä}}{S}=E\cdot\frac{\Delta x}{x}$

# –°–∏–ª–∞ —Ç—Ä–µ–Ω–∏—è
